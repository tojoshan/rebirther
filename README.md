# Droid Tycoon Super Rebirth Tracker

Una aplicación web de acompañamiento diseñada para celular en paralelo para seguir los requisitos de **Super Rebirth (Ciclo 3, Niveles 1 al 23)** del juego Droid Tycoon en Fortnite.

## 🔗 URL del Proyecto

Puedes acceder a la aplicación desde tu dispositivo móvil o computadora en:
👉 **[https://tojoshan.github.io/rebirther/](https://tojoshan.github.io/rebirther/)**

## 🚀 Características
* **Desplazamiento corregido:** Al seleccionar tu nivel de Rebirth actual $N$, la aplicación muestra los requisitos exactos para avanzar al Rebirth $N+1$.
* **Recomendaciones inteligentes:** Cada tarjeta te indica dinámicamente si debes **Vender** (cuando el droide ya no es requerido en ningún nivel futuro), **Conservar** (sirve para niveles próximos) o **Mejorar** (requerido para el Rebirth inmediato), detallando la meta máxima de la guía para evitar gastos innecesarios de recursos.
* **Diseño optimizado para móvil:** Grilla de 2 columnas de ancho, botones táctiles grandes aptos para pulgares, scroll vertical fluido y selector de Rebirth con cinta deslizable horizontal.
* **Persistencia local:** Los progresos se guardan en el navegador (`localStorage`) para que no los pierdas entre sesiones de juego.
